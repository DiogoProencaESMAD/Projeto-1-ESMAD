# Projeto-1-ESMAD
## Huebly
Huebly is an accessibility-focused web app built to support people with Color blindness. It features real-time camera assistance for color detection, interactive quizzes, daltonism tests, an integrated chat, and achievement-based progress tracking to make learning and daily color interaction more accessible and engaging.
